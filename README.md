# ~/.dotfiles

Configuration files for `bash`, `zsh`, `vim`, `git`, and more. Compatible with
Linux and OS X environments.

## Installation

Installation requires `bash`.

    git clone git://github.com/sidwood/dotfiles.git ~/.dotfiles
    cd ~/.dotfiles
    ./install.sh

## Uninstall

Dotfiles? What dotfiles?

    cd ~/.dotfiles
    ./uninstall.sh
    cd -
    rm -rf ~/.dotfiles
